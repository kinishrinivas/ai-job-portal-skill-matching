# AI-Powered Job Portal with Skill Matching

## Project Overview
A full-stack web application that uses AI to match job seekers with relevant job postings based on their resume skills.

## Tech Stack
- **Frontend:** React (plain React + CSS)
- **Backend:** Python Flask
- **Database:** MongoDB Atlas
- **AI/NLP:** Python (resume skill extraction)
- **Authentication:** JWT

## Features
- User authentication (Student, Company, Admin)
- Resume upload and AI-based skill extraction
- Job posting by companies
- Intelligent job-resume matching
- Role-based dashboards

## Folder Structure
